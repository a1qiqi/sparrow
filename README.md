# Sparrow 
![sparrow.png](./source/assets/sparrow.png)

## Installation
```
1. git clone http://192.168.13.21/zhengh/sparrow.git
2. cd sparrow
3. npm install
4. gulp
5. Open browser with http://localhost:5001
```
